# Leitor de Arquivos Assíncrono (Node.js)

Este é um utilitário de linha de comando simples desenvolvido em Node.js para praticar a leitura de arquivos de texto. O script utiliza o módulo nativo `fs` para realizar operações de I/O de forma não bloqueante.

> ###  Fins de Estudo
> **Aviso:** Este repositório foi criado exclusivamente para **fins didáticos e de estudo pessoal**. O objetivo é documentar o aprendizado sobre o ecossistema Node.js, manipulação de arquivos e fluxos assíncronos em JavaScript.

## 🚀 Funcionalidades

- Leitura de arquivos utilizando codificação UTF-8.
- Processamento assíncrono para garantir performance e evitar bloqueio da thread principal.
- Validação de argumentos via linha de comando (`process.argv`).
- Tratamento de erros detalhado (ex: arquivos inexistentes ou permissões negadas).

## 📋 Pré-requisitos

Para rodar este script, você precisará de:
* [Node.js](https://nodejs.org/) (versão 12 ou superior)
* [Git](https://git-scm.com/)
